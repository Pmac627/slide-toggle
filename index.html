Slide Toggle widget

Demo 1- This demo initializes the object without passing configuration settings. var st1 = new SlideToggle('st1');It uses CSS transitions to animate the overlay and fallsback to javascript based animation. Also please note that:
  • the overlay closes by clicking anywhere on the body outside of the overlay.
  • pushes the content down
